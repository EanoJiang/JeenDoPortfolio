# 照片数组自动更新工具

这个工具可以自动扫描 `photo` 目录中的所有图片文件，并更新 `script.js` 中的照片数组内容。

## 使用方法

1. 确保已安装 Node.js
2. 把新照片添加到 `photo` 目录下
3. 在命令行中运行:

```bash
node update_photos.js
```

## 功能特点

- 自动扫描 photo 目录下的所有图片文件
- 自动从文件名生成标题(会去除开头的数字和点，结尾的数字，以及扩展名)
- 保持 script.js 文件其他部分不变，只更新照片数组部分
- 支持的图片格式: jpg, jpeg, png, gif, webp

## 注意事项

- 请确保 `script.js` 文件中包含 `const photoData = [` 和对应的结束标记 `];`
- 对于特殊字符的文件名，可能需要手动调整生成的标题
